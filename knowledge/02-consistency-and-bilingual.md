# 跨章节一致性 + 中英文双语一致性核查清单

> 港股招股章程英文版与繁体中文版「同等有效」（equally authentic），二者数字与语义
> 必须完全对应，这是港股最易出错且后果最严重的区域之一。

## 一、跨章节数值一致性
- [ ] Summary 的收入/毛利/净利/资产 = Financial Information 正文 = Accountants' Report
- [ ] MD&A（Financial Information 章节）叙述引用的金额、增长率、margin = 报表/Notes
- [ ] Key financial ratios 表 = 据报表重算值
- [ ] 前五大客户/供应商金额及占比：正文与表格一致，占比合计≤100%
- [ ] 分部收入按产品/地区/客户多口径拆分，各自合计相等且 = 总 Revenue
- [ ] Indebtedness、Capital expenditure、Contingent liabilities 各处一致
- [ ] 各期「期初/期末」在不同表衔接一致

## 二、股权与所得款项用途一致性
- [ ] 各股东持股数之和 = 已发行股本；持股比例之和 = 100.00%
- [ ] 发行后股权结构与发售规模、超额配股权（Over-allotment）口径一致
- [ ] Controlling Shareholder 直接+间接持股各处一致，且穿透计算相符
- [ ] Use of proceeds 各用途金额之和 = 估计净筹资额；各处（Summary/Future Plans/封面）一致
- [ ] 不同发售价区间（low/high end）对应的净额、用途金额分别勾稽

## 三、时间、编号、名称一致性
- [ ] Track Record Period 表述全文统一（如 "the three years ended 31 December 20XX and
      the X months ended 30 June 20XX"）
- [ ] Latest Practicable Date、成立日期、重组日期各处一致
- [ ] 章节号、表号、图号、脚注号连续不重复
- [ ] 交叉引用 "see/詳見 …第 X 節 / page X" 指向正确
- [ ] 公司全称/简称、子公司、关联方名称全文统一（首次定义后一致使用）

## 四、中英文双语逐项核对（关键）
- [ ] **数字完全一致**：两版所有金额、百分比、股数、天数、日期逐一对应
- [ ] **币种与单位一致**：HK$ / 港元、RMB / 人民币、US$ / 美元；'000 / 千元、million / 百万 对应
- [ ] **风险因素语义对应**：英文 Risk Factors 与中文「風險因素」逐条对应、无增减
- [ ] **定义对应**：Definitions / 釋義 中每个 Defined Term 的中英文一一对应
- [ ] **合约/法律条款语义一致**：关连交易、不竞争承诺、对赌/特殊权利等关键条款两版无歧义
- [ ] **人名/公司名/产品名**两版一致（中文繁体、英文拼写）
- [ ] **表格结构一致**：两版表格行列、合计、脚注一致
- [ ] **简繁/口径**：确认中文版为繁体中文，无简体混入；专有名词译名统一

## 五、近终稿专项扫描
- [ ] 残留 `[●]` / `[•]` / `[REDACTED]` / 空白下划线 / 高亮待填项（定价、日期、份数等）
- [ ] "[•] Application Forms"、"[REDACTED]" 在 PHIP/Application Proof 阶段属正常，
      但临近 final prospectus 应已填实——按文件阶段判断并提示
- [ ] 目录页码 = 正文实际页码（中英文两版各自核对）
